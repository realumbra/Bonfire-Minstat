List of Minstat Runners

As stated on the main page, Challengers are those who complete their minstat runs with added disadvantages, such as No Roll/Block/Parry, +0 weapons or No Death. And Champions are those who complete a default minstat run without any added disadvantage. All games including Sekiro and Elden Ring are listed here.

In this secondary page, you can check every runners and their respective runs. Some of them also include content past the specific run that awarded them the title, and might be good examples if you want to learn and improve on your own skills.

Challenger Runners

Below, the Challengers of the server are listed. Challengers was an old system Bonfire utilized to display users and their "Challenger" runs which were Champion runs with an added challenge. This system has since been removed, however all users who completed these runs and their runs will continue to be displayed here indefinitely. You can click on their pictures to be sent to their respective runs and/or videos. Reminder: some runs are old, and guidelines may have changed since. Always check the main page, and ask the moderators if you have any doubt. If you'd like to be added to the Champion list you must have submitted a run to Ashnessiah#0010 on The Bonfire. If you leave the server your run(s) will be removed. User avatars will only be changed in bulk when we have the time or upon request from the user.

A Veteran Runner is defined as someone who has been knowingly running in the community for a long time and has also contributed to the community. These contributions can range from written guides to helping new users with their runs. If you have a case to be made that you are a Veteran Runner, please message Ashnessiah#0010 on Discord.
